# CookMyBots X Bot (X Bot)

This is an X/Twitter bot scaffold generated by CookMyBots.

✅ Polling happens inside THIS bot (mentions/search)  
✅ All X API calls go through CookMyBots X Proxy (/proxy) for billing + auth  
✅ AI agents supported via CookMyBots AI Gateway  

## Run
```
npm install
npm run dev
```

## Connect X
In the CookMyBots UI, use **Connect X** for this bot.
